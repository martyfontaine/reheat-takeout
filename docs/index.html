Reheat

Google Takeout strips your photos. Reheat puts the metadata back.

Download for macOS Reheat.dmg · latest release · Apple Silicon & Intel

Free & open source · macOS 12+ · no account, no network

Install

  1. Open Reheat.dmg and drag Reheat onto Applications.
  2. First launch: right-click Reheat → Open → Open (it's an unsigned build — see below).
  3. Install exiftool if you haven't: brew install exiftool.
  4. Drop a Google Takeout .zip into the ~/Reheat folder it makes.

Unsigned build — Gatekeeper

This build isn't code-signed/notarized yet, so macOS warns on first open. Right-click → Open, or run:
xattr -dr com.apple.quarantine /Applications/Reheat.app

Verify your download

Each release publishes its SHA-256 — in the release notes and as a Reheat.dmg.sha256 file beside the download. Check yours against it:

shasum -a 256 Reheat.dmg
